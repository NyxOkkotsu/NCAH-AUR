🐾 Command Guide (How to Control Me!)

Don't leave me waiting, master! Control NCAH using these simple cute parameters:

## 1. Stalking Database (-Ss)

Look up and sniff out matching packages from the remote database layout:


```text
./ncah -Ss [query]
```

Example: ./ncah -Ss fetch

## 2. Deep Inspecting Metadata (-Si)

Reveal the absolute detailed profile, git repositories, votes, and secrets of a bestie package:

```text
./ncah -Si [package_name]
```

Example: ./ncah -Si cachyos-mesh

## 3. Adopting & Installing (-S)

Review the PKGBUILD safety signatures, accept the warnings, and deploy the application to your drive meww~:

```text
./ncah -S [package_name]
```

Example: ./ncah -S 8fetch

## 4. Ghosting & Deleting (-R)

Banish a package completely and wipe its memory from our cozy machine structure:

```text
./ncah -R [package_name]
```

Example: ./ncah -R cachyos-mesh
